### My cool [ohmyzsh](https://ohmyz.sh/) themes

To install, copy the theme file to the OMZ themes directory:

```
cp THEMENAME.zsh-theme ~/.oh-my-zsh/themes/
```

and set the theme in `~/.zshrc`:

```
ZSH_THEME="THEMENAME"
```
